## Кратко описание

TVSeriesApp е уеб приложение за управление на информация за сериали, актьори и епизоди. Системата предоставя пълни CRUD операции и поддържа три нива на потребителски достъп.

## Технологичен стек

- ASP.NET Core 7.0 MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
